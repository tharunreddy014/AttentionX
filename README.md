# AttentionX
🎬 AttentionX Lite – AI Clip Generator
🚀 Problem Statement
Long-form videos like lectures, podcasts, and mentorship sessions contain valuable insights.
However, today’s audience prefers short-form content like reels and shorts.
Manually finding and editing these highlights is time-consuming and inefficient.

💡 Solution
AttentionX Lite is a simple AI-powered tool that automatically converts long videos into short clips.

It uses speech-to-text and timestamps to identify important parts of a video and generate short clips with captions.

⚙️ Features
📤 Upload video
🧠 Automatic transcription
✂️ Clip extraction
📝 Caption overlay
🎥 Generate short videos
🛠️ Tech Stack
Python
Streamlit
MoviePy
Whisper AI
